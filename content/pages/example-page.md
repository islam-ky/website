This is an example page created through the Simple Pages collection. 

## How to Use

1. Go to the CMS admin panel
2. Click on "Simple Pages" in the sidebar
3. Click "New Simple Pages"
4. Fill in the title, description, and content
5. In "Menu Settings", configure how it appears in the navigation:
   - **Menu Name**: What appears in the menu (leave blank to use page title)
   - **Weight**: Order in menu (lower numbers first, e.g., 10, 20, 30...)
   - **Parent Menu**: To make it a dropdown item, enter parent like "about-us"
   - **Identifier**: If this page will have sub-items, give it a unique ID

## Features

- Clean, simple layout
- Automatically added to navigation menu
- Full markdown support
- SEO-friendly with title and description
